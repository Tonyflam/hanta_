# hanta_

## Project direction

The original idea was to launch a meme-driven social account and token around **hantavirus**.

This repository now reframes that idea into a **safe and responsible awareness campaign**:

- Do **not** promote disease panic, misinformation, or financial speculation tied to public health fears.
- Do **not** run “pump/trend” token tactics or tax/liquidity mechanics designed to exploit hype.
- Focus on educational, transparent, and ethical communication.

## Practical, safer plan

1. Create an X account focused on **hantavirus awareness**, prevention, and verified updates.
2. Use memes carefully for reach, but keep all health claims sourced and accurate.
3. Add clear disclaimers: not medical advice, no emergency guidance, link to health authorities.
4. If monetization is needed, prefer transparent paths (sponsorships/donations/merch), not speculative token schemes.
5. Establish moderation rules for misinformation and harmful content.

## Research checklist (before launching anything public)

- Verify local legal requirements for health communications and promotions.
- Review platform policies for medical misinformation and deceptive financial promotion.
- Prepare a source list (WHO/CDC/local public health authority) and citation workflow.
- Define crisis response steps for viral misinformation posts.

## Goal

Build a project that can trend **without harming people**: accurate information, ethical messaging, and clear transparency.
